## 格式转换
```
ros2 bag info mine_mapping_001.bag
# 得到(/home/gy/ROS2_Coming/02bag/DOC.txt)

# bag转db3
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple rosbags


rosbags-convert --src mine_mapping_001.bag --dst ros2_db3dir

# db3转pcd
colcon build --symlink-install  --cmake-args -DCMAKE_BUILD_TYPE=RelWithDebInfo -DCMAKE_EXPORT_COMPILE_COMMANDS=1

source ./install/setup.bash

ros2 launch rosbag2_to_pcd rosbag2_to_pcd.launch.xml
```
